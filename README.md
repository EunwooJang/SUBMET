# SUBMET

SUBMET is a Python-based toolkit for processing and analyzing data, featuring scripts to handle various aspects of data processing, calibration, and visualization.

## Features

- **Main File**:
  - `main.py`: Core script to handle data processing and analysis.

- **Directory Generator**:
  - `directory_generator.py`: Script to create necessary directory structures.

- **Example Scripts**:
  - `ex_raw_to_datavolt.py`: Converts raw data into DataVolt format.
  - `ex_raw_to_header.py`: Extracts header information from raw data.

  - `ex_extract_calibration_info_from_datavolt.py`: Extracts calibration information from DataVolt files.
  - 
  - `ex_calibrate_uncalibrated_datavolt.py`: Calibrates uncalibrated DataVolt data using calibration method using SUBMET
  - `ex_revert_datavolt_to_uncalibrated.py`: Reverts DataVolt data to an uncalibrated state.

  - `ex_plot_noisy_event.py`: Plots noisy events.
    
  - `ex_extract_noise_sample.py`: Extracts noise samples from data.
  - `ex_plot_find_ratio_simulation.py`: Visualizes ratio simulation results.
  - `ex_noise_region_detect.py`: Detects noise regions in the data.
    
  - `ex_plot_noise_detection_result.py`: Plots noise detection results.
  - `ex_plot_noise_included_channels_set.py`: Visualizes channels with noise inclusion.
